Description
===============
ぺったんRは、ブラウザひとつで誰でも簡単に漫画を作成できるシステムです。
作成された漫画はプラグインによってブログやSNSなどに表示することも可能です。
ぺったんRプロジェクトは様々なwebシーンに漫画を登場させることでコミック文化を
広く世界に拡げていくことを目標としています。
